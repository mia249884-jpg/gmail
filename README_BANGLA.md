# টেলিগ্রাম ওটিপি বট (Telegram OTP Bot) - সেটআপ গাইড

এই বটটি আপনার জিমেইল (Gmail) থেকে অটোমেটিক ওটিপি (OTP) সংগ্রহ করে টেলিগ্রামে পাঠিয়ে দেয়। এটি গুগল শিটের সাথেও যুক্ত থাকে যাতে ইউজার ডাটা সেভ করা যায়।

---

## ১. জিমেইল কানেক্ট করার পদ্ধতি (Gmail Connection)

এই বটটি জিমেইল থেকে ইমেইল পড়ার জন্য **IMAP** প্রোটোকল ব্যবহার করে। আপনার জিমেইল কানেক্ট করতে নিচের ধাপগুলো অনুসরণ করুন:

### ক. জিমেইল সেটিংস পরিবর্তন:
1. আপনার জিমেইল অ্যাকাউন্টে লগইন করুন।
2. **Security** সেকশনে যান এবং **2-Step Verification** চালু করুন।
3. এরপর **App Passwords** সার্চ করুন বা খুঁজে বের করুন।
4. একটি নাম দিন (যেমন: `TG Bot`) এবং **Create** এ ক্লিক করুন।
5. আপনি ১৬ অক্ষরের একটি পাসওয়ার্ড পাবেন। এটি কোথাও কপি করে রাখুন।

### খ. কনফিগারেশন আপডেট:
আপনার কাছে দুইভাবে জিমেইল যোগ করার সুযোগ আছে:
- **ফাইলের মাধ্যমে:** `config.json` ফাইলটি ওপেন করুন। `gmail_list` এ আপনার জিমেইল দিন এবং `imap_passwords` সেকশনে জিমেইল ও অ্যাপ পাসওয়ার্ডটি লিখে দিন।
- **বটের মাধ্যমে:** বটটি চালু করার পর `/admin` কমান্ড দিন। এরপর **"Gmail যোগ"** বাটনে ক্লিক করে জিমেইল এবং অ্যাপ পাসওয়ার্ড প্রদান করুন।

---

## ২. বটটি চালানোর পদ্ধতি (How to Run)

বটটি সফলভাবে চালানোর জন্য নিচের ধাপগুলো অনুসরণ করুন:

### ধাপ ১: প্রয়োজনীয় লাইব্রেরি ইনস্টল করা
আপনার কম্পিউটারে পাইথন ইনস্টল করা থাকলে, ফোল্ডারের ভেতর কমান্ড প্রম্পট (CMD) ওপেন করে নিচের কমান্ডটি দিন:
```bash
pip install -r requirements.txt
```

### ধাপ ২: গুগল শিট সেটআপ (Google Sheet Setup)
বটটি ইউজার ডাটা সেভ করার জন্য গুগল শিট ব্যবহার করে।
1. [Google Cloud Console](https://console.cloud.google.com/) এ যান।
2. একটি প্রজেক্ট তৈরি করে **Google Sheets API** এবং **Google Drive API** এনাবল করুন।
3. একটি **Service Account** তৈরি করে তার **JSON Key** ডাউনলোড করুন।
4. এই ফাইলটির নাম পরিবর্তন করে `credentials.json` রাখুন এবং বট ফোল্ডারে সেভ করুন।
5. একটি নতুন গুগল শিট তৈরি করুন এবং তার টাইটেল `config.json` এর `sheet_title` (যেমন: "Alias Bot Data") এর সাথে মিল রাখুন।
6. শিটটি আপনার সার্ভিস অ্যাকাউন্টের ইমেইলের সাথে শেয়ার করুন (যাতে বট লিখতে পারে)।

### ধাপ ৩: বট কনফিগারেশন
`bot.py` ফাইলের ভেতর বা `config.json` এ আপনার নিজের টেলিগ্রাম বট টোকেন এবং অ্যাডমিন আইডি সেট করুন।
- `TELEGRAM_TOKEN`: আপনার বটের টোকেন।
- `admin_ids`: আপনার টেলিগ্রাম আইডি (সংখ্যায়)।

### ধাপ ৪: বট রান করা
সবকিছু ঠিক থাকলে নিচের কমান্ডটি দিন:
```bash
python bot.py
```

---

## ৩. প্রধান কমান্ডসমূহ

- `/start`: বট শুরু করা এবং নিজের আলিয়াস (Alias) ইমেইল পাওয়া।
- `/gen`: নতুন ওটিপি ইমেইল তৈরি করা।
- `/admin`: অ্যাডমিন প্যানেল ওপেন করা (শুধুমাত্র অ্যাডমিনদের জন্য)।

---

**সতর্কতা:** জিমেইল এর সাধারণ পাসওয়ার্ড এখানে কাজ করবে না, অবশ্যই **App Password** ব্যবহার করতে হবে। গুগল শিটের জন্য `credentials.json` ফাইলটি ছাড়া বটটি ডাটা সেভ করতে পারবে না।
